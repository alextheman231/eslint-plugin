This is a new minor release of the `@alextheman/eslint-plugin` package. It introduces new features and other backwards-compatible changes that should require little to no refactoring. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Remove the TypeDoc configuration.
  - It is not needed here as it's only ever used in one other repository in the organisation.
  - It is also not expected to be used anywhere else as of now as this one was always intended to be more for my own internal use rather than public use, hence the minor release.
- Replace eslint-plugin-import with eslint-plugin-import-x
  - This appears to be a fork of eslint-plugin-import, but it also supports ESLint v10 unlike eslint-plugin-import.
  - This will give us one less peer dependency warning to worry about while we wait for the React ESLint plugins to catch up.
